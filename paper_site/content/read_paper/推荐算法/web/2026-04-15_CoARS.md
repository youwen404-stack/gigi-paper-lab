## What this paper is about

这篇论文把“推荐”显式改写成一个多轮交互过程：一个 recommender agent 与一个 user agent 反复对话、逐步澄清偏好并给出推荐（agentic recommender systems, ARS）。作者认为现有 ARS 多依赖“把历史轨迹存成文本记忆并检索进 prompt”的 Reflexion 风格做法，经验没有内化到参数里，导致模型更多在用通用推理，而不是逐渐学会推荐决策能力。

为此作者提出 **CoARS**：一个用于“共同演化”的 self-distilled RL 框架，同时优化 recommender agent 与 user agent。

## Main method (high level)

CoARS 由两条互补的学习机制组成：

1. **Interaction reward（交互奖励）**
   - 从同一条交互轨迹中，为 recommender 与 user 两个 agent 构造“耦合的任务级监督信号”，强调双方在交互中相互影响、共同塑造轨迹（而不是只看单方或只看最终结果）。

2. **Self-distilled credit assignment（自蒸馏式 credit assignment）**
   - 把历史轨迹转成 token-level 的 credit 信号，通过 teacher-student conditioning 生成更稠密的监督，让学习不再只依赖最终回报。

## Claimed contributions / results

- 提出一个面向 ARS 的自蒸馏 RL 训练范式，显式利用交互过程产生的“内生监督”。
- 相比代表性 ARS baselines，在多个数据集上报告了更好的推荐效果与更高的 user alignment。

## Quick takeaway

一句话：把 agentic 推荐里的“多轮交互轨迹”变成可学习、可内化的监督（任务级 + token 级），让 recommender 与 user agent 在 RL 中共同进化，而不是只靠外置记忆 + 通用推理。
